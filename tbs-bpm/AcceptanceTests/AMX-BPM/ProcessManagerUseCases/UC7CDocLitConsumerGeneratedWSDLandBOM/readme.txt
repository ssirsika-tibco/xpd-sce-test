UAT Project:  UC7CDocLit
------------------------------------------------------------
This UAT is used to test the generation of WSDLs of Document-
Literal types when the "Generate" button on the Web Service 
task is used.

* The WSDL is generated as required. 
* This project uses both simple types & BOM references.
____________________________________________________________

Important Note:: WSDL is generated using the Service task (not the service provider process) & the process formal parameter refers to both Simple & complex types

WSDL Document literal 
------------------------------------------------------------
* An element is created for each parameter and all the parts
	are element references.
