UAT Project:  UC7DocLitSimpleTypes
------------------------------------------------------------
This UAT is used to test the generation of WSDLs from the process. 
The process uses only simple types. 
* All types are tested in this project.
* No Business Object models are used in this project.
____________________________________________________________


Important Note:: WSDL is generated for the service provider process & the process formal parameter refers to only Simple types

WSDL Document literal 
------------------------------------------------------------
* An element is created for each parameter and all the parts
	are element references.
