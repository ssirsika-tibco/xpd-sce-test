This package specifies the XPDL-to-BPEL converter API. To convert the file "xpdlFile" to the output directory "destFolder", the calling code would look like this: 
 
XPDL2BPEL converter = new XPDL2BPEL(xpdlFile);
IStatus status = converter.convertToBPEL(destFolder.getFullPath().toOSString());
if (!status.isOK()) {
    throw new CoreException(status);
}
