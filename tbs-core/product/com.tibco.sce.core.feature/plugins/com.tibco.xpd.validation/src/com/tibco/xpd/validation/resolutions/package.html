This package contains the base resolution classes that developers will need to create new resolutions.

If a custom marker is defined that extends the base com.tibco.xpd.validation.ValidationMarker then a marker resolution should be extended for this marker type (if quick fixes are required for any issues that will raise a problem with this marker type). Define extension org.eclipse.ui.ide.markerResolution and use com.tibco.xpd.validation.resolutions.GenericResolutionGenerator as the generator class; this class uses the extension com.tibco.xpd.validation.resolution to build the quick fixes for validation issues.
